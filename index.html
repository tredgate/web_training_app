Tredgate Webtrain

Welcome to the Tredgate Webtrain! This is a set of pages designed to help you with test automation. You can find various examples of different web elements and their interactions.

Content of the pages:

  • Home Page  - This page. Provides various of content type which can be identified
  • Registration Form  - A form with various input types, including text, email, and password fields.
  • Web Actions  - A page demonstrating various web actions like drag and drop, double click, and mouse hover.
  • ARIA Testing  - A page demonstrating ARIA attributes and their usage in web applications.
  • Dynamic Content  - A page with dynamic content generation and size testing.
  • Contact  - A simple form for user inquiries.

Any data you enter in the forms will not be sent anywhere. This is a static test page for training purposes only.
